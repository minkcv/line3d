# Line 3D
Draw wireframe meshes in the browser. Requires WebGL

[minkcv.github.io/line3d/](https://minkcv.github.io/line3d/)

### Features
- Create, delete, connect, disconnect, and move points.
- Rotate, zoom, and pan an orthographic camera.
